# Simple GraphQl server

## Setitng it up

1. MongoDB

    **Mac**
    ```
    brew install mongodb && mongod
    ```

    **Ubuntu**
    ```
    sudo apt-get -y install mongodb
    ```

2. NodeJs

    From [NodeJS Website](https://nodejs.org/en/)

    or in **Mac**

    ```
    brew install nodejs
    ```

3. Clone repo
4. Install deps by

    ```
    npm install
    ```
    or
    ```
    yarn install
    ```
5. Load dummy data

    ```
    npm run dummy
    ```

6. Start server
    ```
    npm start
    ```

7. Go to [localhost:3000/graphql](http://localhost:3000/graphql) and have fun
